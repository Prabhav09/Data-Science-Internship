## 🔹 Task 03 – Exploratory Data Analysis (EDA) on Titanic Dataset
In this task, I performed a **Mini Exploratory Data Analysis (EDA)** to extract deeper insights.

### Data Cleaning
- Filled missing **Age values using mean**
- Dropped irrelevant columns such as **Cabin**
- Created **Family Size feature**

### Analysis Performed
- Survival rate by **Age Group**
- Survival rate by **Embarkation Port**
- Survival rate by **Family Size**

### Visualizations
- Age Distribution Histogram
- Correlation Heatmap
- Survival by Family Size Bar Plot

### Tools Used
Python, Pandas, Matplotlib, Seaborn
