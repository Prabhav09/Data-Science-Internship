# 📊 Task 03 – Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Objective
The goal of this task is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to discover deeper survival patterns using data cleaning techniques and advanced visualizations.

---

## 📊 Dataset
Titanic Dataset (Kaggle)

The dataset contains passenger information such as:
- Age
- Gender
- Passenger Class
- Embarkation Port
- Number of Family Members
- Survival Status

---

## 🔧 Data Cleaning
The dataset was cleaned using the following techniques:
- Filled missing values in the **Age** column using the **mean value**
- Dropped irrelevant columns such as **Cabin**
- Created a new **Family Size** feature using `SibSp + Parch`

---

## 🔍 Analysis Performed
The following questions were analyzed:

- Survival rate by **Age Group**
- Survival rate by **Embarkation Port**
- Survival rate by **Family Size**

---

## 📈 Visualizations Created
- Histogram for **Age Distribution**
- **Correlation Heatmap**
- **Bar Plot** for Survival by Family Size
