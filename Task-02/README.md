# 🚢 Data Science Internship – Task 02  
## Titanic Dataset Analysis & Visualization

---

## 📌 Task Objective
The objective of this task was to analyze the famous Titanic dataset and identify survival patterns using Python. This task focuses on data cleaning, exploratory data analysis (EDA), and visualization.

---

## 📊 Dataset Used
- Titanic Dataset (Kaggle)
- Contains passenger details such as:
  - Age
  - Gender
  - Passenger Class
  - Survival status

---

## 📝 Task Requirements
1. Load Titanic dataset.
2. Handle missing values (e.g., Age column).
3. Analyze survival patterns:
   - Who survived more: Males or Females?
   - Did passenger class affect survival?
   - What was survival rate by age group?
4. Create visualizations:
   - Bar chart of survival by gender
   - Bar chart of survival by passenger class
   - Histogram of passenger ages

---

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook
- Kaggle

---

## 📈 Key Findings
- Female passengers had a higher survival rate compared to males.
- First-class passengers had better survival chances.
- Survival varied across different age groups.

## 📂 Files Included
- Titanic dataset (.csv)
- Jupyter Notebook (.ipynb)
- Visualization outputs

- Handling missing data
- Exploratory Data Analysis (EDA)
